To operate the Prince HTML SMS to PDF programme follow these steps:

1. Install "Prince HTML SMS to PDF Setup" and enter the LICENCE KEY

2. Either rename the folder in the Extract called 'files' to '_files' OR
   move all the HTML SMS files into a folder called HTML

3. Drag all the HTML SMS files into the window on the left

4. Drag the "Configuration" file into the CSS window - top right.

5. Click convert



==================================================================
==================================================================


Converting from PDF to JPEG

1. Open Adobe Acrobat Reader (do not open one of the PDF documents,
   Adobe Acrobat Reader should have a grey background)

2. Go to Advanced > Document Processing > Batch Processing...

3. Select the process titled "SMS HTML to JPEG"

4. Press OK

5. Select the PDF files you just converted from Prince

6. Select a destination folder (the EDS/AV FOLDER/RAW)

7. DONE
